# API Security Assignment

## 1. Risks of exposing API keys
Exposing API keys publicly (e.g., on GitHub) can lead to unauthorized usage. Attackers may use the key to send requests, which can result in unexpected charges, service abuse, or even system compromise.

## 2. Why avoid logging city names?
Logging city names can expose user location data, which is considered sensitive information. According to privacy principles like GDPR (data minimization), such data should not be stored or displayed unnecessarily.
